# ActiveStructOpt

Forked from [ActiveStructOpt/update-sampler](https://github.com/akshaydaf/ActiveStructOpt/tree/update-sampler), which is forked from [ActiveStructOpt](https://github.com/Fung-Lab/ActiveStructOpt)
